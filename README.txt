playertracker version 1.2.3

Gets ip address and username and then puts them in ../worldpath/ip_file.txt. Could be useful for server admins.

Written by Lucas Burlingham and NoNameDude and licensed under the MIT License.

There are no dependancies, as this mod is for any game.

Note: Because of the nature of dynamic IP addresses, not all instances of a user will be under one singular IP address, 
though it is possible for one user to have the same IP address for an extensive period of time.
